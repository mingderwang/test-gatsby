# A Gatsby Project
> A Quick Start for Gatsby Project with MDX Plugin and TailwindCSS/DaisyUI.
 
## Refer to and Inspired by
https://www.sitepoint.com/gatsby-mdx-blog/

## Build and Run
```
yarn
```
then
```
yarn dev
```

## Deploy with vercel
```
vercel
```

## Features
* Gatsby
* MDX
* Tailwind
* DaisyUI
* vercel (now)
* Blogs
